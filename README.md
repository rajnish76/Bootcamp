# Bootcamp API

> Backend API for Bootcamp.

## Usage

Setup "config/config.env" with your credentials

## Install Dependencies

```
npm install
```

## Run App

```
node server.js
```

## Database Seeder

To seed the database with users, bootcamps, courses and reviews with data from the "\_data" folder, run

```
# Destroy all data
node seeder -d

# Import all data
node seeder -i
```
